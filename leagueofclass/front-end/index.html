Venha fazer parte desse game do mundo real



O League of Class é uma plataforma de suporte para os professores inspirada no modo mais didático de agradar os alunos: GAMES. Esta é uma ferramenta de aprendizado incrível. Você cadastrar a sua turma gratuitamente e gerencia-la. Além de construir games para uma atividade prática para os seus alunos. Conheça e use LOC. Garantimos que todos irão gostar
